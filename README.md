# Anime365 — расширение для AnymeX / Mangayomi / Dartotsu

Источник [Anime365 (smotret-anime)](https://smotret-anime.online) в формате расширений Mangayomi (JavaScript). Работает на Windows, Android, Linux, macOS, iOS — везде, где работают AnymeX и Mangayomi.

> ⚠️ Для просмотра видео нужен аккаунт Anime365 с **активной подпиской**. Каталог и списки серий доступны без аккаунта.

## Возможности

- Поиск, «Популярное» (по рейтингу) и «Онгоинги»
- Страница тайтла: постер, описание, жанры, статус, оценка MAL
- Список серий с датами выхода
- Выбор перевода: озвучки / субтитры / оригинал, фильтр по языку, лимит на количество
- По умолчанию первой идёт озвучка (а не саб с высоким рейтингом Anime365); настраивается («Что показывать первым»)
- Субтитры не прикрепляются к озвучке, чтобы не включались сами (можно вернуть переключателем)
- Подпись начинается с автора — AnymeX запоминает ту же озвучку при переходе на следующую серию
- Фильтр по минимальному разрешению (например, только 1080p+)
- Софтсаб ASS подключается автоматически
- Зеркала сайта переключаются в настройках

## Установка

1. Установите [AnymeX](https://github.com/RyanYuuki/AnymeX/releases) (или [Mangayomi](https://github.com/kodjodevf/mangayomi/releases)).
2. Добавьте URL репозитория расширений (в AnymeX: профиль → настройки → Extensions; тип — Anime):
   - опубликованный: `https://raw.githubusercontent.com/X4vi3/anime365-extension/main/anime_index.json`
   - локальный (для разработки): запустите `node serve.mjs` и добавьте `http://127.0.0.1:18365/anime_index.local.json`
3. Установите источник **Anime365** из списка.

## Настройка аккаунта

В настройках источника Anime365 достаточно указать **e-mail** и **пароль** от аккаунта с активной подпиской — расширение само обменяет их на бессрочный `access_token` (запрос `/api/login`) и закэширует его.

Поле **app** уже заполнено готовым идентификатором API-клиента (это публичный `client_id`, не секрет) — менять его не нужно. Если хотите свой: создайте API-клиент на `https://<ваше-зеркало>/api-clients` и впишите полученное значение.

Поле **Access token** оставьте пустым. Оно нужно только если вы хотите входить без e-mail и пароля — токен можно получить в браузере, где вы вошли на сайт: `https://<зеркало>/api/accessToken?app=<app>`.

## Структура

- `javascript/anime/src/ru/anime365.js` — код расширения
- `anime_index.json` — индекс репозитория (его URL добавляется в приложение)
- `anime_index.local.json` + `serve.mjs` — вариант для локального тестирования

## API

Расширение использует официальное публичное API сайта (`/api`, OpenAPI-спека есть на `/api-docs`):
`/series` (каталог, поиск), `/series/{id}` (тайтл + серии), `/episodes/{id}` (переводы серии), `/translations/embed/{id}` (ссылки на видео и субтитры, требует токен), `/login` (получение токена).

## Известные проблемы

- **AnymeX 3.0.9 (стабильная) на Windows: чёрный экран при живом звуке.** Это баг плеера самого AnymeX (его сборка не отображает H.264 High profile — то есть 1080p/720p с Anime365; 360p в Baseline играет). Решение: ставить **бета-версию AnymeX** — там плеер починен. Mangayomi не подвержён.
- «No servers available» при живом каталоге обычно означает ошибку логина — проверьте app/email/пароль в настройках источника (после переустановки расширения они сбрасываются).

## История версий

- **0.2.4** — озвучка по умолчанию первой (раньше топовый по рейтингу саб запускался сам); субтитры больше не прикрепляются к озвучке; автор в начале подписи — AnymeX запоминает озвучку между сериями. Настройки «Что показывать первым» и «Субтитры поверх озвучки».
- 0.2.3 — фильтр по минимальному разрешению (1080p+ и т.д.) и сортировка вариантов (по качеству / по популярности Anime365). Низкие разрешения скрываются из списка.
- 0.2.2 — устойчивое чтение настроек: фикс «Error when getting source preference» в AnymeX на Android (чтение незаполненных настроек роняло воспроизведение). Все настройки теперь имеют жёсткие значения по умолчанию.
- 0.2.1 — поле `app` предзаполнено, порядок настроек: сначала e-mail/пароль; поле токена явно помечено необязательным
- 0.2.0 — стабильный релиз (то же, что 0.1.3)
- 0.1.3 — автоматический перелогин, если сервер отверг закэшированный access_token
- 0.1.2 — нормализация типографских апострофов/кавычек в поиске (названия из AniList приходят с U+2019)
- 0.1.1 — названия по умолчанию в ромадзи: без этого автоподбор тайтла в AnymeX (фаззи-сравнение с AniList) не находит источник
- 0.1.0 — первая версия

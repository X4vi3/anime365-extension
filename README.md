# Anime365 — расширение для AnymeX / Mangayomi / Dartotsu

Источник [Anime365 (smotret-anime)](https://smotret-anime.online) в формате расширений Mangayomi (JavaScript). Работает на Windows, Android, Linux, macOS, iOS — везде, где работают AnymeX и Mangayomi.

> ⚠️ Для просмотра видео нужен аккаунт Anime365 с **активной подпиской**. Каталог и списки серий доступны без аккаунта.

## Возможности

- Поиск, «Популярное» (по рейтингу) и «Онгоинги»
- Страница тайтла: постер, описание, жанры, статус, оценка MAL
- Список серий с датами выхода
- Выбор перевода: озвучки / субтитры / оригинал, фильтр по языку, лимит на количество
- Качество 1080/720/480, софтсаб ASS подключается автоматически
- Зеркала сайта переключаются в настройках

## Установка

1. Установите [AnymeX](https://github.com/RyanYuuki/AnymeX/releases) (или [Mangayomi](https://github.com/kodjodevf/mangayomi/releases)).
2. Добавьте URL репозитория расширений (в AnymeX: профиль → настройки → Extensions; тип — Anime):
   - опубликованный: `https://raw.githubusercontent.com/X4vi3/anime365-extension/main/anime_index.json`
   - локальный (для разработки): запустите `node serve.mjs` и добавьте `http://127.0.0.1:18365/anime_index.local.json`
3. Установите источник **Anime365** из списка.

## Настройка аккаунта

1. Один раз создайте API-клиент на странице `https://<ваше-зеркало>/api-clients` (понадобится войти на сайт). Полученный идентификатор — это `app`.
2. В настройках источника Anime365 укажите: **app**, **e-mail**, **пароль**.
   Расширение само обменяет их на бессрочный `access_token` (запрос `/api/login`) и закэширует его.
3. Альтернатива: вставьте `access_token` напрямую (поле «Access token») — тогда логин/пароль не нужны. Токен можно получить в браузере, где вы вошли на сайт: `https://<зеркало>/api/accessToken?app=<ваш app>`.

## Структура

- `javascript/anime/src/ru/anime365.js` — код расширения
- `anime_index.json` — индекс репозитория (его URL добавляется в приложение)
- `anime_index.local.json` + `serve.mjs` — вариант для локального тестирования

## API

Расширение использует официальное публичное API сайта (`/api`, OpenAPI-спека есть на `/api-docs`):
`/series` (каталог, поиск), `/series/{id}` (тайтл + серии), `/episodes/{id}` (переводы серии), `/translations/embed/{id}` (ссылки на видео и субтитры, требует токен), `/login` (получение токена).

## Известные проблемы

- **AnymeX 3.0.9 (стабильная) на Windows: чёрный экран при живом звуке.** Это баг плеера самого AnymeX (его сборка не отображает H.264 High profile — то есть 1080p/720p с Anime365; 360p в Baseline играет). Решение: ставить **бета-версию AnymeX** — там плеер починен. Mangayomi не подвержён.
- «No servers available» при живом каталоге обычно означает ошибку логина — проверьте app/email/пароль в настройках источника (после переустановки расширения они сбрасываются).
